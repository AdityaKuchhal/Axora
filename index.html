System Requirements

  • Operating System: Windows 10 or later
  • Python: Python 3.8 or higher (if using script version)
  • Storage: 500MB free space recommended
  • Note: Standalone executable available from GitHub Releases (no Python required)

Quick Start Guide

  • Option 1 (Recommended): Download the standalone executable from GitHub Releases - no installation required
  • Option 2: Clone the repository and run the Python script after installing dependencies
  • Prepare your Excel mapping file with provider, corporation, and account information
  • Select your source folder and destination folder in the application
  • Execute and let Axora organize your files automatically

About Axora

Built for professionals who value efficiency

Axora was developed to address the challenge of managing utility bills across multiple corporations and providers. What used to require hours of manual organization now happens automatically with intelligent automation.

Why Choose Axora?

  • Time Saving: Reduce bill organization time by 90%
  • Error Free: Eliminate manual sorting mistakes
  • Professional: Clean, consistent folder structures
  • Scalable: Handle any number of files and folders
  • Reliable: Built with enterprise-grade PyQt6 framework
  • Secure: All processing happens locally on your machine
